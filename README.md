# 忍狐

忍狐のホームページです。[ninko.neocities.org](https://ninko.neocities.org) で公開しています。

## 開発環境構築手順

```
$ git clone https://github.com/ninko-txz/ninko.git
$ cd ninko
$ npm init
$ npm run serve
```

and open http://localhost:8080

## Deploy

```
$ NEO_API_KEY=xxxxxxxxxx ./deploy.sh
```
